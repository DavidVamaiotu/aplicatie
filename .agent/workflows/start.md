// turbo-all

1. Read `AGENTS.md` at the repository root to load repository-specific instructions.
2. Read `progress.md` at the repository root to understand the current state of the project.
   - If `progress.md` does not exist, note that it needs to be created during the first code-changing task.
3. Read any relevant skill files referenced in `AGENTS.md` if the current task matches their description.
4. Proceed with the user's request, following the rules in `AGENTS.md`.
5. After any code changes, update `progress.md` immediately with what changed and why.
